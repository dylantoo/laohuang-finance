    📊 量价关系 — 完整学习手册¶

    量是因，价是果。 成交量是市场的"能量"，价格是市场的"方向"。看懂量价关系，就拿到了读懂K线的钥匙。

    一、核心认知：量价底层逻辑¶

    1.1 成交量是什么？¶

    成交量 = 一段时间内市场参与者用真金白银投票的总数

    • 放量 → 多空分歧加大（有人疯狂买，有人疯狂卖）
    • 缩量 → 多空一致（大家都观望，或都认为该涨/该跌）
    • 天量 → 极端情绪，往往是转折点
    • 地量 → 交投冰点，往往是底部区域

    1.2 量价关系的基本原理¶

    原理 解释
    量在价先 成交量变化先于价格变化出现。主力建仓时不会拉价格，但量会先放大
    量价同步 涨时放量、跌时缩量 = 健康趋势
    量价背离 涨时缩量、跌时放量 = 趋势难以持续
    天量天价 成交量创阶段性天量时，价格离顶不远
    地量地价 成交量萎缩到极致时，价格见底

    1.3 量价分析的核心问题¶

    看任何一根K线（或一段走势），问自己三个问题：

    ① 价格在涨还是跌？
② 成交量是放大还是缩小？
③ 当前价格在什么位置？（低位/中位/高位）

    三个问题组合起来，就是量价分析的完整框架。

    二、量价八大经典形态¶

    2.1 价涨量增（最健康）¶

    描述 量价同步上涨
    图形 📈 价格↑ + 成交量柱↑
    含义 需求旺盛，新资金持续进场，趋势有望延续
    操作 ✅ 持仓/加仓（只要没到高位）
    注意 连续大涨后出现 ➡ 可能接近BC（买入高潮）

    适用场景：主升浪、突破关键阻力位时

    2.2 价涨量缩（背离预警）¶

    描述 价格在涨，成交量却在萎缩
    图形 📈 价格↑ + 成交量柱↓
    含义 上涨动能减弱，跟风盘不足，顶部信号
    原因 ①主力已控盘，少量资金就能拉（缩量上涨最可怕，因为没人卖）
    ②多头力竭，买盘后继无力（危险）
    操作 ⚠️ 减仓观望，不要追高

    关键判断： - 缩量上涨+连续创新高 → 主力高度控盘，见顶前还有一段 - 缩量上涨+冲高回落 → 大概率见顶

    2.3 价涨量平（趋势延续）¶

    描述 价格在涨，成交量保持稳定
    图形 📈 价格↑ + 成交量柱→
    含义 趋势正常延续，供需平衡
    操作 ✅ 继续持有

    适用：趋势中段，主力已完成建仓，持筹稳定

    2.4 价跌量增（恐慌信号）¶

    描述 价格下跌，成交量放大
    图形 📉 价格↓ + 成交量柱↑
    含义 抛压沉重，市场恐慌
    两种可能 ① 高位价跌量增 = 主力出货，必须离场
    ② 低位价跌量增 = SC（恐慌抛售），最后一跌
    操作 高位 ➡ ❌ 坚决离场
    低位 ➡ ⚠️ 观察是否出现SC+Spring组合

    关键判断：位置决定性质。低位恐慌放量往往是机会，高位恐慌放量是灾难。

    2.5 价跌量缩（缩量阴跌）¶

    描述 价格下跌但成交量萎缩
    图形 📉 价格↓ + 成交量柱↓
    含义 抛压减轻，杀跌动能不足，底部区域信号
    操作 ✅ 关注底部反转形态出现后介入
    注意 缩量阴跌也可能是下跌中继（如果均线仍空头排列）

    口诀：缩量下跌还会跌，缩量止跌才是底。

    2.6 价跌量平（无量下跌）¶

    描述 价格下跌，成交量持平
    图形 📉 价格↓ + 成交量柱→
    含义 市场麻木，没人恐慌也没人抄底
    操作 ⚠️ 观望，等待放量信号出现

    2.7 价平量增（变盘前兆）¶

    描述 价格横盘，成交量放大
    图形 ➡ 价格→ + 成交量柱↑
    含义 多空博弈激烈，方向即将选择
    位置判断 低位横盘+放量 → 吸筹（主力在建仓）
    高位横盘+放量 → 派发（主力在出货）
    操作 等待方向确认后再行动

    2.8 价平量缩（自然调整）¶

    描述 价格横盘，成交量萎缩
    图形 ➡ 价格→ + 成交量柱↓
    含义 多空都休息，市场在等待催化剂
    位置判断 高位横盘+缩量 → 可能变盘向下
    低位横盘+缩量 → ST（二次测试），吸筹确认
    操作 低位缩量横盘 ➡ ✅ 关注突破机会

    三、威科夫量价分析（VSA）¶

    源自《威科夫操盘法》孟洪涛著，是目前最系统、最实战的量价分析体系。

    3.1 量价四象限图（快速扫一眼就懂当前状态）¶

                       成交量↑（放量）
                       │
        ⚠️ 放量下跌     │     ✅ 放量上涨
        恐慌/出货/SC    │     SOS/需求旺盛/健康
        ───────────────┼───────────────→ 价格↑
        🔵 缩量下跌     │     🟡 缩量上涨
        供应枯竭/筑底    │    动能减弱/背离预警
                       │
                   成交量↓（缩量）
    象限 状态 操作 颜色
    右上 放量上涨（SOS） ✅ 持有/加仓 🟢
    右下 缩量上涨（背离） ⚠️ 减仓 🟡
    左上 放量下跌（SC或SOW） ⚠️ 减仓/观望 🔴
    左下 缩量下跌（供应枯竭） ✅ 关注反转 🔵

    3.2 七大关键信号（必须牢记）¶

    ① SOS — Sign of Strength（强势信号）¶

    特征 放量突破阻力位，收盘在阻力上方
    K线 中大阳线，实体饱满，没有长上影
    量 成交量 > 20日均量 1.5倍以上
    位置 吸筹完成后、整理平台突破
    操作 JOC确认后 → 加仓

    ② JOC — Jump Over the Creek（越过小溪）¶

    描述 放量突破前期高点/颈线/下降趋势线
    量 突破日成交量显著放大
    确认 突破后缩量回踩不破 → true JOC
    操作 JOC确认 → 🚀 主升浪启动点

    核心意义：JOC是买盘力量大于供应（卖盘）的终极证据。阻力位就是"小溪"，跳过去说明多方全胜。

    ③ BC — Buying Climax（买入高潮）¶

    特征 天量阳线但涨幅有限
    量 创阶段/历史天量
    K线 实体很小（甚至可能是十字星/小阳线）
    位置 连续大涨后的高位
    含义 最后一波散户接盘，主力正在派发
    操作 ⚠️ 3-5天内大概率见顶，减仓

    BC的典型心理： 

    前期：缩量上涨（主力控盘，不卖）
高潮日：放巨量，还涨了点 → 散户终于忍不住追了
结局：主力把货倒给了散户，后面开始跌

    ④ UT — Upthrust（上冲回落/假突破）¶

    特征 突破阻力位后收盘回到阻力位下方+放量
    量 放量（说明有人借突破出货）
    K线 长上影线或阴线覆盖
    含义 假突破，诱多陷阱
    操作 ❌ 空仓不追，持仓者减仓

    UT vs JOC 关键区别：

    特征 ✅ JOC（真突破） ❌ UT（假突破）
    收盘价 在阻力位上方 回到阻力位下方
    成交量 放量+量>1.5倍均量 放量但可能是天量
    之后走势 缩量回踩不破 直接大跌回去
    上影线 不长 长上影

    ⑤ SC — Selling Climax（恐慌抛售）¶

    特征 暴跌+天量，最后一波恐慌盘涌出
    量 创阶段天量（比之前所有下跌日都大）
    K线 长下影线（V形反转当天）
    位置 长期下跌后的低位
    含义 最恐慌的人割肉了，卖压耗尽
    操作 ✅ 关注，但不等SC就抄底。等二次测试

    SC的经典形态： 

    价格开盘大跌→散户恐慌抛售→量急剧放大→
尾盘被主力接起→留下长下影线

    ⑥ Spring — 弹簧效应¶

    特征 击穿支撑位后快速收回+放量
    量 放量（说明有人在这个位置接货）
    K线 长下影线，收盘在支撑位上方
    位置 SC之后或吸筹末期
    含义 回调完毕的最强确认信号
    操作 ✅ 入场信号

    为何叫"弹簧"：

    价格被压到支撑以下（像压弹簧），然后弹起来 → 压得越狠（放量穿透），弹得越高。

    SC + Spring 组合（最经典的底部确认）： 

    ① 暴跌放量（SC）  →  恐慌抛售
② 缩量阴跌  →  自然调整
③ 再次砸破低点但迅速收回+放量（Spring）  →  上车！

    ⑦ ST — Secondary Test（二次测试）¶

    特征 回调到前期低点/支撑位，但成交量大幅缩小
    量 明显缩量（< SC天量的1/3）
    位置 SC之后、Spring之前
    含义 供应枯竭，没有人再愿意卖
    操作 ✅ ST是吸筹确认信号，准备入场

    3.3 威科夫吸筹三段论（完整底部结构）¶

    第一阶段：初次供应（PSY）
  价格还在跌，但已有聪明钱开始小量买入
  量：偶尔放量

第二阶段：恐慌抛售（SC）
  暴跌+天量，最后一批恐慌盘出逃
  主力在此大量接货

第三阶段：二次测试（ST）+ 弹簧（Spring）
  ST：缩量回踩，供应枯竭确认
  Spring：放量砸穿支撑后收回 → 上车！
      │
      ▼
  JOC：放量突破颈线 → 主升浪启动

    3.4 威科夫派发三段论（完整顶部结构）¶

    第一阶段：初始派发（PSY）
  高位放量滞涨，聪明钱开始出货

第二阶段：买入高潮（BC）
  天量阳线但涨幅小，散户蜂拥接盘

第三阶段：UTAD（上冲回落后的派发）
  UT假突破 → 缩量反弹 → 再次放量下跌
  最终跌破颈线 → 确认顶部

    四、量价背离实战¶

    4.1 顶背离（卖出信号）¶

    特征 价格创新高，但成交量比前高缩量
    级别 越大的周期背离越可靠（周线 > 日线）
    可靠性 连续2次顶背离 → 见顶概率极高
    操作 减仓，设止盈

    图解： 

    价格： 低点A → 高点B（更高） → 高点C（更高）
量：   放量       缩量            缩量更小
        ↑         ↑                ↑
       正常      ⚠️ 背离1         🔴 背离2→走

    4.2 底背离（买入信号）¶

    特征 价格创新低，但成交量比前低缩量
    级别 越大的周期越可靠
    可靠性 连续2次底背离 → 底部概率极高
    操作 关注，结合Spring确认后入场

    图解： 

    价格： 高点A → 低点B（更低） → 低点C（更低）
量：   放量       缩量            缩量更小
        ↑         ↑                ↑
       正常      ⚠️ 背离1         🟢 背离2→关注

    4.3 量价背离的止损意义¶

    顶背离 ≠ 立刻跌，可能是主力高度控盘下的缩量加速阶段。 - 顶背离+长上影线 → 大概率见顶 - 顶背离+连续小阳线 → 可能还有缩量加速段 - 底背离 ≠ 立刻涨，可能还有最后一跌

    4.4 量价背离的周期联动¶

    周期 多周期共振规则
    周线顶背离 + 日线顶背离 🔴 大级别顶部，必须减仓
    周线底背离 + 日线底背离 🟢 大级别底部，重点关注
    日线顶背离 + 60分顶背离 ⚠️ 短期调整，减半仓
    日线底背离 + 60分底背离 ✅ 短线反弹机会

    五、分时图量价解读¶

    5.1 分时量价的核心看什么¶

    维度 看什么 含义
    分时均价线 价格在均线上方还是下方 均线上方 = 强势，下方 = 弱势
    分时量柱 红绿柱高低 红柱高=主动买盘多，绿柱高=主动卖盘多
    早盘30分钟量 是否放量 决定全天基调
    尾盘30分钟量 是否抢筹/砸盘 对第二天有指向意义

    5.2 典型的强势分时¶

    时间 形态 量价特征
    9:30-10:00 高开高走 第1个5分钟放量（换手龙），价格站稳均线上方
    10:00-11:30 高位横盘 缩量整理，均价线不破
    13:00-14:30 再次放量上攻 下午放量突破上午高点
    14:30-15:00 高位收盘 不放量跳水，收在全天高位

    特征总结：早盘放量上攻 → 缩量整理 → 午后放量再攻 → 尾盘高位收盘

    5.3 典型的分时陷阱¶

    诱多分时： 

    早盘  快速拉升 → 放量（散户追）
中盘  震荡走低 → 缩量（散户等反弹）
午后  再次冲高 → 放量但不过前高（双头）
尾盘  跳水 → 放量下跌（主力出货完毕）

    诱空分时： 

    早盘  快速下杀 → 放量（散户割肉）
中盘  震荡企稳 → 缩量（没人卖了）
午后  稳步回升 → 温和放量（主力吸筹）
尾盘  拉到均线上方 → 全天深V

    5.4 分时盘口量价速查¶

    盘口现象 量价解读 应对
    开盘半小时量>昨日全天30% 换手充分，今日有行情 关注方向选择
    涨时放量、跌时缩量（分时） 主力在吸筹 持有或关注
    涨时缩量、跌时放量（分时） 主力在出货 减仓
    尾盘放量拉升 抢筹（利好预期）或诱多 次日验证
    尾盘放量跳水 抢跑（利空预期） 减仓
    全天缩量横盘 主力休息，方向未定 观望

    六、不同位置看量价¶

    同样的量价形态，在不同位置含义完全不同。

    6.1 低位（底部区域）¶

    形态 含义 操作
    放量下跌（SC） 恐慌最后一跌 关注，不抄底
    缩量阴跌 抛压减轻，还需等待 观望
    缩量横盘（ST） 吸筹基本完成 ✅ 重点关注
    放量反弹后缩量回踩 上涨前的回测 ✅ 上车点
    放量突破（JOC） 底部确立，主升启动 ✅ 加仓

    6.2 中位（上涨途中）¶

    形态 含义 操作
    放量上涨 趋势延续，资金持续进场 ✅ 持有
    缩量回调 正常洗盘，浮筹洗净 ✅ 加仓
    放量滞涨 可能有阻力，应者云集但打不开空间 ⚠️ 观察
    缩量上涨 动能减弱，可能接近顶部 ⚠️ 减仓

    6.3 高位（顶部区域）¶

    形态 含义 操作
    天量阳线（BC） 买入高潮，即将见顶 ❌ 减仓
    放量下跌 主力出货或恐慌开始 ❌ 离场
    放量冲高回落（UT） 假突破诱多 ❌ 必须走
    缩量反弹 反弹无力，下跌中继 ❌ 减仓
    缩量横盘 多空僵持，变盘向下概率大 ⚠️ 减仓

    6.4 关键位置速查¶

    低位 + 放量杀跌 = SC（恐慌抛售）
低位 + 缩量横盘 = ST（二次测试，吸筹完成）
低位 + 放量突破 = JOC（主升启动信号）

中位 + 放量上涨 = 健康趋势（持有）
中位 + 缩量回踩 = 正常洗盘（加仓）
中位 + 放量滞涨 = 遇到阻力（警惕）

高位 + 天量阳线 = BC（买入高潮）
高位 + 放量假突破 = UT（诱多陷阱）
高位 + 放量下跌 = 开始出货（离场）

    七、量价综合判断框架¶

    7.1 五步量价分析法¶

    Step 1: 看位置 → 低位/中位/高位
Step 2: 看方向 → 涨/跌/横盘
Step 3: 看量能 → 放量/缩量/平量
Step 4: 看确认 → 是否有威科夫信号（BC/SC/UT/Spring/JOC）
Step 5: 做决策 → 买入/持有/减仓/离场

    7.2 量价情绪温度计¶

    量价状态 颜色标识 含义
    放量上涨 🟢 健康，需求旺盛
    缩量上涨 🟡 上涨背离，动能减弱
    天量滞涨 🔴 BC，顶部区域
    放量下跌 🟣 恐慌/出货，危险
    缩量下跌 🔵 供应枯竭，关注底部
    缩量横盘 ⚪ 方向未定，观望
    放量突破 🟢 JOC确认，主升启动
    缩量反弹 🟠 反弹无力，下跌中继

    7.3 量价 + 技术指标组合判断¶

    量价状态 叠加指标 结论 置信度
    放量上涨 RSI<70 主升浪健康 ★★★★★
    放量上涨 RSI>80 可能到顶 ★★★
    缩量上涨 RSI>70 顶背离确认 ★★★★★
    放量下跌 RSI<20 SC恐慌抛售 ★★★★
    缩量下跌 RSI<20 底背离可能 ★★★★
    放量突破 EMA金叉 JOC+均线共振 ★★★★★
    缩量回踩 EMA趋势向上 正常洗盘 ★★★★

    7.4 成交量基准（量比）¶

    量比 含义 解读
    <0.5 极度缩量 交投冰点，底部特征
    0.5-0.8 缩量 观望情绪浓厚
    0.8-1.2 正常量 趋势正常延续
    1.2-1.5 温和放量 资金关注增加
    1.5-2.5 明显放量 多空分歧加大
    2.5-5 巨量 重大事件驱动
    >5 天量 极端情绪，大概率转折

    量比 = 今日成交量 / 过去20日平均成交量

    八、常见误区与陷阱¶

    ❌ 误区1：放量就追¶

    真相：放量可能是主力在出货（UT/BC），也可能是主力在吸筹（SC/JOC）。放量不是买入理由，放量的位置和K线形态才是。

    ❌ 误区2：缩量就一定见底¶

    真相：缩量下跌可能是下跌中继（在下跌趋势中，缩量只是说明没人买，不是说明卖完了）。缩量止跌+放量回升才是见底。

    ❌ 误区3：天量天价是一回事¶

    真相：天量不等于当即天价。天量出现后3-5天内才见顶。BC之后主力还有余货要出，往往会高位震荡一周左右。

    ❌ 误区4：价跌量缩就是洗盘¶

    真相：主力洗盘通常是缩量，但缩量下跌也可能是无人接盘的阴跌。区别在于： - 洗盘：缩量跌到关键支撑位就弹起来（有人护盘） - 阴跌：缩量跌破支撑位（没人管，自由落体）

    ❌ 误区5：只看日线量价¶

    真相：大周期（周线/月线）上的量价信号比日线可靠得多。周线JOC比日线JOC重要10倍。

    ❌ 误区6：量价万能论¶

    真相：量价要结合大盘环境、板块走势、市场阶段一起看。大盘环境好时量价信号更准，环境差时容易假信号。

    🐛 陷阱1：对倒放量¶

    主力自己左手倒右手制造放量假象。常见于： - 底部放量但股价不动（诱多建仓） - 高位放量但K线十字星（出货/对倒维持人气） - 放量突破但第二天就低开低走

    如何识别：看分时图是否锯齿状、是否有大单持续对倒、盘口是否有明显的"托单"和"压单"同时出现。

    🐛 陷阱2：缩量涨停¶

    缩量涨停比放量涨停更强势，说明没人卖，主力高度控盘。但要注意： - 底部缩量涨停 → 反转信号 ✅ - 高位缩量涨停 → 可能开板后瀑布杀 ⚠️

    🐛 陷阱3：利好放量=追¶

    利好公布后的放量，如果是高开低走（放量阴线），说明利好出尽，主力借利好出货。利好公布不涨反跌 + 放量 = 最危险的信号。

    九、实战案例集¶

    案例1：底部放量突破（JOC + Spring）¶

    背景：股价从20元跌到12元，跌幅40%
过程：
  Day 1-3：连续放量暴跌，日成交从1亿→5亿（SC）
  Day 4-10：缩量阴跌至11.5元，成交缩至1亿（供应枯竭）
  Day 11：砸穿11元支撑后V形反转，收12元，放量3亿（Spring）
  Day 12-15：小阳线推升，温和放量
  Day 16：放量突破13元阻力位（JOC），量6亿 → 🚀

判断：底部确立，主升浪启动
操作：Spring确认后建仓30%，JOC确认后加仓至60%

    案例2：高位天量见顶（BC + UT）¶

    背景：股价从10元涨到30元（3倍涨幅）
过程：
  Day 1-3：加速上涨，日成交放大
  Day 4：跳空高开冲34元回落收31元，成交创天量（BC）
  Day 5：低开低走收28元，放量（主力出货）
  Day 6：缩量反弹30元（反弹无力）
  Day 7：放量跌破28元 → 🛑 颈线破位 → 清仓

判断：BC + 放量下跌 + 缩量反弹 = 标准见顶
操作：BC日减半仓，放量跌破28日线清仓

    案例3：量价背离逃顶¶

    背景：上涨趋势中
对比两次冲顶：
  第一波：从20→30元，量从1亿放大到5亿（健康）
  第二波：从28→34元，量从3亿缩到2亿（⚠️ 背离）
  第三波：从32→35元，量从2亿缩到1亿（🔴 背离确认）

判断：连续两次量价顶背离 = 动能耗尽
操作：第二次背离减半仓，第三次背离清仓
结果：第三次背离后股价跌回28元

    案例4：缩量洗盘识别¶

    背景：股价从10元涨到18元后回调
特征：
  Day 1-2：放量冲高18元后回落（放量了，有人卖）
  Day 3-5：缩量跌回15元，量能萎缩到启动前的水平
  Day 6-7：缩量横盘在15元附近，成交量极度萎缩
  Day 8：放量突破前高18元（JOC），成交量5倍于洗盘期

判断：缩量回调不破关键位 = 洗盘不是出货
操作：缩量企稳15元时加仓，JOC突破18元加仓
洗盘 vs 出货区别：
  ✅ 洗盘：缩量、不破位、形态紧凑
  ❌ 出货：放量、破位、形态散乱
