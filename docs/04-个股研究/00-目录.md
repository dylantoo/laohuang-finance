## 个股分析模板

每次分析按以下结构：
1. **宏观方向** → 利好/利空/中性
2. **周期位置** → 所处阶段
3. **错价判断** → 被低估/高估/合理
4. **策略匹配** → 价值/成长/动量
5. **偏误警示** → 认知偏误检查
6. **交易计划** → 入场/止损/止盈/仓位
