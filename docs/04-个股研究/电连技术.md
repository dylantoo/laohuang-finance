# 电连技术（300679）— 完整分析

> 最后更新：2026-04-28
> AI服务器高速连接器龙头，Pogopin全球核心厂商

---

## 一、逻辑摘要

电连技术（300679）是国内射频连接器龙头，通过全资子公司爱默斯成为全球Pogopin（弹簧针）核心厂商。其核心增长引擎不是消费电子，而是**AI服务器高速连接器**——224Gbps PAM4量产，批量供应英伟达、浪潮、新华三、昇腾等头部厂商。

### 核心逻辑链

```
AI算力军备竞赛 → GPU算力提升 → 信号传输速率暴增(56G→112G→224Gbps)
  → 传统BTB连接器不够用 → Pogopin替代 → 百亿级市场爆发(20亿→100亿+)
  → 电连技术(含爱默斯)是国内唯一能批量供货224Gbps的厂商
  → 量价齐升 + 国产替代 + 客户粘性
```

---

## 二、核心优势（3点）

1. **技术壁垒极高**：224Gbps PAM4量产，模具精度2μm，间距0.4mm，寿命10万次+。国内唯一，全球仅3-5家。
2. **客户卡位精准**：英伟达、浪潮、新华三、昇腾——AI服务器全链条覆盖。连接器认证周期长、替换成本高，粘性极强。
3. **国产替代红利**：价格比海外低20-40%，定制周期2-4周（海外8-12周），性价比碾压。

---

## 三、最大风险

> **224Gbps 良率/产能爬坡不及预期**。新技术在量产初期的良率波动是正常现象，但如果持续低于预期，短期股价会承压。

**对策**：止损放在40元（今日低点43.46下方），防的就是这个。
