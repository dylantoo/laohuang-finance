## 📋 操作计划（针对浮盈3%持仓）

### 建议：先减半锁利

**为什么减半：**
1. 3个点浮盈在混沌期已经是该落袋的级别
2. 连续2天放量下跌（72亿→109亿），抛压在放大
3. FIB 23.6%（175.20）已破，下一个支撑在164.98（还有-4.5%）
4. 如果不减，放量杀到164，3%浮盈变成-3%亏损

### 剩余持仓管理规则

| 情景 | 操作 |
|:----|:-----|
| 跌到**167**（成本线附近） | 再减一半，剩25% |
| 跌到**164**（FIB 38.2%） | 全部清仓，不等 |
| 反弹到**178~180** | 全清，落袋 |
| 放量跌破**163** | 全部走 |

### 什么时候可以重新接回

必须同时满足三个条件：
1. **缩量**：日成交额萎缩到50~60亿（109亿的一半）
2. **企稳**：出现长下影线Spring，不再创新低
3. **支撑位**：跌到164~157区间确认有效

三个条件缺一不可——缩量+阴跌不行，到支撑位+放量也不行。
