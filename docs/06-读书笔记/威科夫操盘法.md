## 一、书籍定位

本书是**威科夫理论（Wyckoff Theory）** 的中文权威实战解读版。作者孟洪涛是美国华尔街专业机构交易员，曾任GE Capital风控总监、美国银行投资总监，将威科夫理论应用于美股、期货和中国股指期货市场。

威科夫与道氏、江恩、艾略特并称为**技术分析四大经典理论**。其核心思想是：**市场是由大资金（聪明钱/CM）操纵的，价格行为是供求关系的直接体现**。

## 二、威科夫理论三大基石

### 基石1：市场三定律

| 定律 | 内涵 | 实战意义 |
|:----|:-----|:---------|
| **供求定律** | 供大于求→价格跌；求大于供→价格涨 | 量价分析的根本依据 |
| **因果定律** | 任何大行情都有漫长的准备阶段（吸筹/派发） | 横盘时间越长，后续行情越大 |
| **努力与结果定律** | 成交量（努力）与价格变动（结果）应成正比 | 放量滞涨=异常，放量突破=确认 |

### 基石2：大资金（CM/Clever Money）行为模式

- CM是市场的**主动方**，散户是**被动方**
- CM的目标：**低价买入→高价卖给散户**
- CM的操作流程：**吸筹→洗盘→拉升→派发**
- CM利用市场心理：**恐惧（低位卖出）和贪婪（高位买入）** 获利

### 基石3：量价分析（VSA — Volume Spread Analysis）

威科夫理论中，**量（成交量）和价（价格波幅/Spread）** 是解读CM意图的两种工具：
- **价格**回答"CM做了什么"
- **成交量**回答"CM做得有多坚决"

## 三、威科夫核心工具与信号

### 工具1：看盘顺序

```
① 看趋势方向 → ② 看支撑阻力质量 → ③ 看成交量确认 → ④ 看速度变化 → ⑤ 看收盘位置
```

威科夫强调：不要只看裸K，要有序分析。

### 工具2：五大关键市场行为

| 行为 | 英文 | 含义 |
|:----|:-----|:-----|
| **吸筹** | Accumulation | CM隐蔽买入，价格横盘或缓慢上涨 |
| **派发** | Distribution | CM隐蔽卖出，价格横盘但成交量放大 |
| **震仓** | Shakeout | CM故意打压价格吓走散户，收集筹码 |
| **终极震仓** | Ultimate Shakeout (UT) | 吸筹结束前的最后一次猛烈打压 |
| **弹簧效应** | Spring | 价格击穿支撑位后快速拉回→CM买入信号 |
| **上冲回落** | Upthrust (UT) | 价格突破阻力位后快速回调→CM卖出信号 |
| **吸筹完毕** | No Supply / SOS | 下跌缩量=供应枯竭；放量突破=需求进场 |
| **派发完成** | No Demand / SOW | 上涨缩量=需求不足；放量跌破=供应进场 |

### 工具3：CM操盘完整流程

```
吸筹阶段                → 拉升阶段               → 派发阶段
┌──────────────────┐   ┌──────────────────┐   ┌──────────────────┐
│ PS→SC→AR→ST→LPS  │→→│ LPS→BU→Markup→   │→→│ PSY→BC→AR→ST→LPSY│
│ 初支→震仓→反弹→   │   │ 二次吸筹→真正上涨 │   │ 初支→终极震仓→   │
│ 横盘→二次吸筹     │   │                  │   │ 反弹→横盘→二次派发│
└──────────────────┘   └──────────────────┘   └──────────────────┘

PS = Preliminary Support（初次支撑）
SC = Selling Climax（恐慌抛售高潮）
AR = Automatic Rally（自动反弹）
ST = Secondary Test（二次测试，缩量=吸筹确认）
LPS = Last Point of Support（最后的支撑点/进场点）

BU = Back Up（回调确认）
Markup = 主升浪

PSY = Preliminary Supply（初次供应）
BC = Buying Climax（买入高潮→成交量天量）
LPSY = Last Point of Supply（最后的供应点/卖出点）
SOW = Sign of Weakness（弱势信号）
```

## 四、威科夫量价分析实战信号

### 4.1 看涨信号

| 信号 | 图形特征 | 含义 |
|:----|:---------|:-----|
| **Spring（弹簧）** | 价格跌破支撑位后迅速拉回收盘在支撑上方，成交量放大 | CM在低位接货 |
| **SOS（强势信号）** | 放量中阳线突破阻力位，收盘在高位 | 需求进场，上涨确认 |
| **LPS（最后支撑）** | SOS之后的缩量回调，不破前低 | 二次进场的最佳时机 |
| **震仓后缩量** | 大阴线后连续小阳线缩量回升 | 浮动供应已被清除 |
| **吸收（Absorption）** | 高位放量横盘但不跌 | CM在吸收上方卖盘 |
| **JOC（Jump Over Creek）** | 放量突破关键阻力位 | 主升浪启动信号 |

### 4.2 看跌信号

| 信号 | 图形特征 | 含义 |
|:----|:---------|:-----|
| **UT（上冲回落）** | 价格突破阻力位后迅速回落收盘在阻力下方 | CM在高位抛售 |
| **SOW（弱势信号）** | 放量阴线跌破支撑位 | 供应进场 |
| **BC（买入高潮）** | 成交量创天量但涨幅有限 | 见顶信号 |
| **UTAD** | UT之后的再次测试，成交量放大但无法突破 | 确认派发 |

### 4.3 最关键的量价关系矩阵

| 价格 | 成交量 | 含义 |
|:----|:------|:-----|
| ↑上涨 | ↑放量 | ✅ **健康上涨**，需求推动 |
| ↑上涨 | ↓缩量 | ⚠️ **上涨乏力**，需求不足，可能见顶 |
| ↓下跌 | ↑放量 | ⚠️ **恐慌下跌**，供应充足，可能继续跌 |
| ↓下跌 | ↓缩量 | ✅ **下跌枯竭**，供应耗尽，可能见底 |
| 横盘 | ↑放量 | ⚠️ **CM派发**或**吸筹**，需结合位置判断 |
| 横盘 | ↓缩量 | ✅ **调整结束**，方向即将选择 |

## 五、VS 现有交易体系对照

### 威科夫 vs 主线龙头战法

| 维度 | 主线龙头战法 | 威科夫操盘法 | 融合价值 |
|:----|:-----------|:------------|:---------|
| **核心逻辑** | 选主线+抱龙头 | 读懂CM意图+量价分析 | 威科夫提供了**买在吸筹末端、卖在派发初期**的精确工具 |
| **趋势判断** | EMA金叉死叉 | 供求关系+量价确认 | 威科夫的SOS/SOW信号比EMA更早捕捉趋势启动 |
| **回调判断** | 五维校验法 | Spring+震仓+缩量测试 | 威科夫的Spring是更精确的"回调完成"信号 |
| **见顶判断** | 涨停减少+补涨扩散 | BC+UT+UTAD | 威科夫的BC信号比"补涨扩散"更早指示顶部 |
| **仓位管理** | 阶段仓位+波动率调仓 | 根据供求强度 | 可以结合：**缩量吸筹区重仓，放量派发区减仓** |
| **入场点** | 低吸+突破追 | LPS（最后支撑点） | 威科夫的LPS比单纯低吸更精确 |
| **止损** | 硬性-10% | 跌破Spring/LPS低点 | 威科夫止损**更贴近关键支撑位** |

### 核心提升点

威科夫理论能给主线龙头战法带来**三个最关键的补充**：

#### ① 精确的入场时机——LPS（最后支撑点）

主线龙头战法说"低吸"，但什么时候算"够低了"？威科夫给的答案是：**LPS是回调缩量、不再出现新卖盘的位置**。指标层面：缩量+不破EMA20+收盘在中位以上。

👉 **可以优化到`stock-pivot`的"回调vs退潮五维校验法"中**

#### ② 精确的见顶信号——BC+UT（买入高潮+上冲回落）

主线龙头战法说"高位放量开始警惕"，威科夫更精确：**BC是日线天量但涨幅很小，UT是突破后立刻回到突破价下方**。出现这两个信号就是明确的卖出点。

👉 **可以优化到`trader-psychology`和`stock-pivot`的离场信号中**

#### ③ 量价关系矩阵——最直观的市场情绪判断

目前`stock-pivot`的"情绪温度计"只统计涨跌比和涨停数，不够精细。威科夫的**量价四象限（放量涨→健康、缩量涨→背离、放量跌→恐慌、缩量跌→衰竭）** 可以直接用在日报情绪判断中。

👉 **可以优化到`stock-pivot`的情绪判断工具中**
