## 复盘模板

每次复盘按以下顺序：
1. **大盘位置** → 当前阶段
2. **主线状态** → 老主线还在吗？新主线出现了吗？
3. **持仓复盘** → 按计划执行了吗？有偏离吗？
4. **操作建议** → 持仓/减仓/止损/换股
5. **明日预判** → 关注什么？做什么？
