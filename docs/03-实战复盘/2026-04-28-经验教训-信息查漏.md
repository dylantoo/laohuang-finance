# 2026-04-28 经验教训：信息查漏复盘

> 关键词：电连技术、基本面核实、多源搜索

---

## 事件

分析电连技术（300679）时，只查了实时行情和K线数据，没有专门搜索公司的核心业务和增长逻辑，导致：

1. 不知道Pogopin是AI服务器高速连接器核心产品
2. 不知道224Gbps量产的关键技术壁垒
3. 不知道英伟达、浪潮、昇腾等核心客户关系
4. 把电连技术误判为"消费电子逻辑"，错过重要行情

---

## 失误链

```
① 只查实时行情 + K线 → 没搜公司基本面
② 看到电子器件板块-1.48% → 误判"没有板块共振"
   （但立讯精密是做消费电子的，电连做的是AI服务器连接器——完全不同的赛道）
③ K线接口被封后 → 没有再尝试其他搜索渠道
   （老黄用豆包搜到了，说明信息是公开的）
```

---

## 改进后的个股分析流程

从今以后，分析个股必须走以下完整流程：

```
▸ 第一步：实时行情 ✅
   新浪API查股价、成交额、振幅、涨跌幅

▸ 第二步：技术指标 ✅
   EMA/RSI/量比/波动率计算

▸ 第三步：基本面核实 ← 这次漏了！
   用browser搜以下至少一个来源：
   ├─ 百度搜公司主营业务、核心产品、增长逻辑
   ├─ 东方财富搜研报、财务数据、竞争对手
   ├─ 搜狗微信/头条搜最新新闻动态
   关键词示例："电连技术 Pogopin 224Gbps"
                       "公司名 核心客户"
                       "公司名 最新研报 2026"

▸ 第四步：交叉验证
   查到的信息与已有知识是否冲突？
   有没有相反的观点？
   老黄提到的逻辑是否成立？
```

---

## 警惕"可得性启发"陷阱

> **不要因为同行业板块有个股在跌，就默认整个板块不行。**

举例：
- 电连技术（AI服务器连接器）vs 立讯精密（消费电子连接器）
- 同一个"电子器件"行业大分类，但细分赛道完全不同
- 一个在跌不代表另一个的逻辑有问题
