# 123冷冻期硬规则

> 针对"看到回调就追进去被套"的解决方案

---

## 规则

当板块涨了30%+开始回调时：

```
1️⃣ 第一天回调 → 别动，观察怎么跌
2️⃣ 第二天回调 → 继续看，会不会反包
3️⃣ 第三天回调 → 如果跌到EMA20企稳了 → 开始考虑上车
```

🚫 **前3天绝对不买！**

---

## 为什么是3天？

- 真回调 → 3-5天企稳反弹
- 假回调（退潮）→ 第4天还在跌，越跌越快

第3天末或第4天初是最安全的观察窗口。

---

## 配合五维校验法

第3天如果跌到EMA20附近企稳了，再跑一遍[05-方法论/回调vs退潮五维校验法](05-方法论/回调vs退潮五维校验法.md)，五个维度中三个以上健康 → 上车。
