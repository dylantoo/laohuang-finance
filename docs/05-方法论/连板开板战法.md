# 连板开板战法 — 什么能追，什么不能追

> **核心认知**：连板股开板不是末日，有些开板是"空中加油"（上车机会），有些是"断头闸刀"（接盘陷阱）。区别在于**量、价、位、时**四维特征。

## 一、核心框架：四维判断法

看任何一个开板，同时问自己四个问题：

```

│            │           │           │         │
```

## 二、可以追的开板（4种）

### 🟢 类型1：缩量换手板（最强形态）

| 特征 | 说明 |
|------|------|
| **形态** | 第2-3板开板放量，随后再次封板 |
| **量** | **缩量回封**（回封时量能比开板时明显小） |
| **分时** | 开板后缓慢回封，分时图流畅，均价线跟上 |
| **位置** | 连板初期（2-4板），**低位板** |
| **含义** | 浮筹换手清洗完毕，主力锁仓，**加速段启动** |

**为啥能追**：
- 开板放出筹码→浮筹走了→缩量回封→说明没人卖了
- 后面往往还有2-3个一字板加速

**操作**：开板震荡时介入，缩量回封确认后加仓

**口诀**：低位换手放量开，缩量回封我就来。

### 🟢 类型2：N字型开板回封

| 特征 | 说明 |
|------|------|
| **形态** | 早盘封板→盘中炸板→震荡→午后回封 |
| **量** | 炸板时放量，回封时缩量 |
| **分时** | 走"门"字形或"N"字型，有明确支撑位 |
| **位置** | 连板中段 |
| **含义** | 主力故意开板洗盘，清除不坚定筹码 |

**为啥能追**：
- 炸板后价格不破分时均线，说明承接强
- 午后回封说明主力还在控盘
- 这种洗盘后次日溢价率高

**操作**：在分时均线附近低吸，或回封时追入

**口诀**：炸板不破均价线，午后回封敢加钱。

### 🟢 类型3：回封+次弱转强（游资最爱）

| 特征 | 说明 |
|------|------|
| **形态** | 烂板（反复开板）→ 尾盘勉强封住 → **次日竞价高开** |
| **量** | 当天量很大（换手充分） |
| **精髓** | **看点不在当天，在看次日竞价** |
| **位置** | 龙头分歧日 |

**为什么要等次日竞价**：
- 烂板当天谁都不知道能不能活
- 烂板后**次日高开3%以上** = 弱转强确认 ✅
- 烂板后**次日低开或平开** = 弱转弱，放弃 ❌

**操作**：当天不追，**次日高开确认弱转强后再追**

**经典案例**：很多妖股在3板、5板位置都出现过烂板+次弱转强的洗盘

**口诀**：烂板烂板不要怕，次日高开就是妈。

### 🟢 类型4：龙头首阴反包（最强回马枪）

| 特征 | 说明 |
|------|------|
| **形态** | 连板过程中某天高开低走收阴线（首阴） |
| **量** | 放量阴线 |
| **次日** | **放量反包涨停** |
| **位置** | 龙头主升浪中段 |

**为啥能追**：
- 主升浪中的第一次阴线 = 分歧，不是见顶
- 次日反包 = 分歧转一致，开启第二波
- **龙头首阴反包是确定性最高的买点之一**

**注意事项**：
- 必须是主升浪中的**第一次阴线**
- 第二次、第三次就不灵了
- 必须放量反包，缩量反包是诱多

**口诀**：龙头首阴我不怕，放量反包大胆上。

## 三、不能追的开板（4种）

### 🔴 类型1：高位一字开板放巨量

| 特征 | 说明 |
|:----:|:------|
| **形态** | 连续一字板后，某天终于开板了 |
| **量** | **历史天量**（换手率>30%甚至50%+） |
| **分时** | 开板后剧烈震荡，无法回封 |
| **位置** | 连板高位（5板+、涨幅翻倍+） |
| **含义** | 一字板无法买入，开板是给你**进场接盘的机会** |

**为什么不能追**：
- 一字板阶段主力已经赚够了
- 开板放巨量 = 主力把筹码倒给散户
- 后面大概率A杀（连续跌停）

**案例**：
```
连续5个一字板 → 第6天开板巨量换手60% → 当天勉强回封
→ 第7天跌停 → 第8天跌停 → A杀
```

**口诀**：一字板开板放天量，谁追谁是接盘侠。

### 🔴 类型2：炸板不回封（断头闸刀）

| 特征 | 说明 |
|:----:|:------|
| **形态** | 封板后被一笔大单砸开，**持续走低收盘不封** |
| **量** | 炸板后持续放量 |
| **分时** | 炸板后**直线跳水**，跌穿分时均线 |
| **位置** | 任何位置都可能出现 |

**为什么不能追**：
- 这说明主力不想封了，甚至主力自己砸的
- 炸板不回封 = **多转空**，趋势改变了
- 次日大概率低开低走

**和N字型开板回封的区别**：

| 特征 | ✅ 能追（N字回封） | ❌ 不能追（断头闸刀） |
|:----:|:----------------:|:------------------:|
| 炸板后走势 | 横盘震荡，不破均线 | 直线跳水，跌破均线 |
| 回封 | 有，且缩量 | 没有，持续走低 |
| 收盘 | 封住涨停 | 收在低位 |
| 次日 | 高开溢价 | 低开低走 |

**口诀**：炸板不封直线跳，管住手来管住脑。

### 🔴 类型3：加速板后再次放量开板

| 特征 | 说明 |
|:----:|:------|
| **形态** | 已经走了3个以上的**缩量加速板**，今天再次开板放量 |
| **量** | 前3天缩量→今天突然放巨量 |
| **分时** | 快速拉板后迅速炸开，反复烂板 |
| **位置** | 高位加速段 |

**为什么不能追**：
- 缩量加速板说明已经没人卖了
- 突然放量 = 筹码松动 = **聪明钱在跑**
- 这是最后的换手，不是洗盘

**判断方法**：
```
第1板：放量 ✅ 合理
第2板：缩量 ✅ 加速
第3板：缩量 ✅ 加速
第4板：再次放量开板 ❌ 危险 → 这是主力在出货
```

**口诀**：加速缩量三连板，再放量开是狼来了。

### 🔴 类型4：板块退潮期个股硬顶

| 特征 | 说明 |
|:----:|:------|
| **形态** | 板块内其他个股都跌了，就这一只还在封板 |
| **量** | 封单很小，硬顶 |
| **分时** | 开板后勉强回封，封单只有几千万 |
| **板块** | 同板块其他股票大跌/跌停 |
| **含义** | **个股成为独狼，板块没了** |

**为什么不能追**：
- 独狼没有板块支撑 = 随时断板
- 龙头死于孤军奋战
- 明天板块不回暖，补跌概率极大

**口诀**：板块退潮独狼撑，开板千万不能碰。

## 四、分时图开板形态速查

### 看分时一眼判断

```
能追的形态（承接好）：
┌─────────────────────────────┐
│ 封板→炸板→横盘不破均线→回封  │
│                            │
│  涨停 ──────┬────┬────      │
│            │    │ 回封      │
│            │ 炸板            │
│  均线 ──────┴────┴────      │
│              ↑              │
│           不破均线 = 强承接  │
└─────────────────────────────┘

不能追的形态（出货）：
┌─────────────────────────────┐
│ 封板→炸板→跳水→跌破均线→不回封│
│                            │
│  涨停 ────┐                  │
│           │ 炸板              │
│  均线 ────┴────────────────  │
│           │                  │
│           ▼ 跳水不回封       │
└─────────────────────────────┘
```

### 六种标准分时形态

| 形态 | 走势描述 | 判断 | 操作 |
|:----:|:--------|:----:|:----:|
| **T字板** | 一字开盘→盘中瞬间开板→秒回封 | ✅ **最强，可以追** | 开板瞬间就是买点 |
| **N字板** | 封板→炸板横盘→回封 | ✅ **可以追** | 回封时追，或均线低吸 |
| **烂板** | 反复开板→尾盘勉强封住 | ⚠️ **等次日确认** | 当天不追，看次日竞价 |
| **天地天** | 涨停→跌停→再涨停 | ⚠️ **游资标的，谨慎** | 非龙头不追 |
| **断头板** | 封板→炸板直线跳水→收盘不封 | ❌ **不能追** | 管住手 |
| **开口板** | 高开→冲板→遇阻回落→收阳线 | ❌ **不能追** | 看次日是否反包 |

## 五、龙头开板特殊场景

### 5.1 龙头分歧转一致（最经典）

**标准剧本**：
```
第1板：放量涨停（分歧）
第2板：缩量加速（一致）
第3板：放量开板→回封（分歧转一致）✅ — 龙头买点
第4+板：开始真的一字加速
```

在第3板分歧转一致时追入，后面还有2-3个板的肉。

### 5.2 龙头首阴（反包买点）

**标准剧本**：
```
阶段1：主升浪5个板
阶段2：第6天高开低走收阴线（首阴）→ ⚠️ 观察
阶段3：第7天放量反包涨停 → ✅ 上车！
阶段4：再走2-3天冲顶
```

### 5.3 龙头断板后反包（N+1板）

**标准剧本**：
```
某龙头走了6板，第7板断板收阴
第8板放量反包涨停 → ✅ 这是"N+1"反包板
```

**N+1反包板的特征**：
- 断板日不能是巨量A杀（要缩量调整）
- 反包日必须放量（>断板日量能的一半）
- 反包日要早盘快速封板（10:30前）
- 午后反包的不可靠

### 5.4 龙头分歧日买点速查表

| 第几板 | 量能特征 | 判断 | 操作 |
|:------:|:--------|:----:|:----|
| 1板 | 放量涨停 | 启动 | 不确定，观望 |
| 2板 | 缩量加速 | 确认龙头地位 | ✅ 可追 |
| 3板 | 放量开板→缩量回封 | **分歧转一致，最经典买点** | ✅ **买入** |
| 4-5板 | 继续缩量加速 | 主升段 | ✅ 持有 |
| 5+板 | 再次放量开板 | 可能是出货，不是洗盘 | ❌ 减仓 |
| 断板 | 首阴+缩量 | 龙头首阴 | ⚠️ 观察 |
| 断板次日 | 放量反包 | 龙头二波 | ✅ 买入 |
| 断板后 | 连续缩量或反包失败 | 龙头终结 | ❌ 放弃 |

## 六、仓位与止损纪律

### 仓位

| 开板类型 | 仓位建议 | 说明 |
|:--------:|:--------:|:------|
| 缩量换手板（低位） | 30-50% | 确定性高 |
| N字回封（中段） | 20-30% | 需要次日确认 |
| 次弱转强（烂板次日高开） | 30% | 次日竞价确认后再入 |
| 龙头首阴反包 | 30-50% | 确定性最高的买点之一 |
| 高位一字开板放量 | **0%** | 绝对不碰 |
| 断头闸刀（炸板不回封） | **0%** | 绝对不碰 |
| 加速后再次放量开 | **0%** | 出货信号 |

### 止损

| 场景 | 止损位 |
|:----|:------:|
| 第二天低开3%以上 | 直接止损 |
| 第二天低开平走不拉 | 半小时内不翻红→止损 |
| 第二天尾盘不涨停 | 减半仓 |
| 买入后3天还在震荡不板 | 走人（时间止损） |
| 炸板后次日跌停 | 跌停板挂单出 |

### 买点铁律

```
❌ 当天炸板的，当天不买（除非确定是缩量回封）
✅ 当天回封确认的，当天可买一部分
✅ 次日竞价确认弱转强的，大胆买
✅ 龙头首阴反包的，开盘确认后买入
✅ N+1反包板的，放量突破昨日高点买入
```

## 七、实战口诀与速查表

### 七字诀

```
一字开板放天量   → 不追（接盘）
低位换手板回封   → 追（换手充分）
炸板不破均价线   → 等回封（承接强）
炸板跌破均价线   → 不碰（筹码散）
烂板烂板看次日   → 高开就追（弱转强）
加速板后再放量   → 跑（主力出货）
龙头首阴反包板   → 大胆追（确定性高）
板块退潮独狼撑   → 放弃（独木难支）
```

### 速查：看一眼就能判断

| 开板类型 | 一句话 | 能追？ |
|:--------:|:------|:-----:|
| 低位换手回封 | 浮筹换完，加速在即 | ✅ |
| N字回封 | 洗盘不破均线 | ✅ |
| 烂板次日高开 | 弱转强确认 | ✅ |
| 龙头首阴反包 | 分歧转一致 | ✅ |
| 高位一字开板放天量 | 主力全出 | ❌ |
| 炸板不回封跳水 | 多转空 | ❌ |
| 加速后再放量开 | 最后换手=出货 | ❌ |
| 独狼硬顶 | 板块没了 | ❌ |

### 段位自测

```
初段：见到开板就冲进去（韭菜）
中段：知道回封才追（新手）
高段：区分回封是真洗盘还是假突破（老手）
顶级：等次日竞价确认弱转强再下手（高手）
```

## 八、案例集

### 案例1：可以追 — 分歧转一致（3板经典）

```
股票：某PCB龙头
第1板：涨停+放量（分歧）
第2板：涨停+缩量（一致，确认龙头）
第3板：高开9%→炸板跌到3%→横盘→尾盘回封
        量：上午炸板放量，下午回封缩量
        分时：不破分时均价线
        判断：分歧转一致 ✅

操作：回封确认后买入30%仓位
结果：后面还有3个涨停板
```

### 案例2：不能追 — 一字板开板放天量

```
股票：某重组概念
第1-5板：全部一字板，无法买入
第6板：开板，换手率55%，成交额创历史天量
        全天反复烂板，尾盘勉强封住
        分时：巨量封单→被砸→再封→再被砸
        判断：主力出货 ✅

操作：不追
结果：次日跌停开盘，3天跌回原点
```

### 案例3：可以追 — 龙头首阴反包

```
股票：某AI龙头
第1-5板：主升浪，5连板
第6板：高开低走收阴线-4%，换手率25%
        盘后判断：龙头首阴，不是顶
第7板：竞价高开3.5%，开盘后15分钟封板
        放量反包 ✅

操作：竞价高开3.5%确认弱转强→开盘买入30%
结果：再走3天，累计再涨25%
```

### 案例4：不能追 — 炸板不回封

```
股票：某芯片概念
第1-3板：连续涨停
第4板：上午封板→11:00被大单砸开
        炸板后直线跳水跌到+1%
        跌破分时均线，全天不回封
        收盘收在+3%
        判断：断头闸刀 ❌

操作：不碰
结果：次日低开-6%，尾盘跌停
```

### 案例5：可以追 — N+1反包板

```
股票：某机器人龙头
阶段1：7连板（翻倍+）
阶段2：第8天断板，收-2%，缩量调整
         判断：不是A杀，正常调整
阶段3：第9天高开4%，放量涨停反包
         判断：N+1反包 ✅

操作：放量过昨日高点时买入
结果：再走2天，累计再涨15%
```

---

> **连板开板的本质是筹码博弈**。开板 ≠ 坏事，关键是看**谁在买、谁在卖、买盘能不能接住卖盘**。
>
> 缩量回封 = 卖压枯竭，买盘大于卖盘 = 能追
> 放量不回封 = 卖盘碾压买盘 = 不能碰
> 次日高开确认 = 弱转强信号 = 最安全
