# 混沌期生存法则

> 无主线、轮动快时怎么操作

---

## 混沌期特征

- 板块轮动速度：2-3天一个
- 龙头封板率：＜50%
- 赚钱效应：今天涨明天跌
- 成交量：缩量

## 三不 + 三可

| ❌ 不做 | ✅ 做 |
|--------|------|
| 不追高 | 轻仓试错（≤30%） |
| 不满仓 | 埋伏等风口 |
| 不做T | 低吸不追涨 |

## 四不碰三要

### ❌ 四不碰
1. 昨天大涨的
2. 涨了2周的
3. 游资纯概念
4. 业绩差估值虚

### ✅ 三要
1. 有政策底、产业逻辑的
2. 跌了3-5天、缩量地量的
3. 龙头走势独立的

## 仓位

| 规则 | 数值 |
|------|------|
| 总仓位 | ≤30% |
| 单票 | 10% |
| 止损 | -5% |
| 现金 | ≥70% |

## 止盈止损

- 浮盈5-8% → 止盈
- 浮亏-5% → 止损

## 核心心态

> 我不是在赚钱，我是在活着等下一个主线的到来。
