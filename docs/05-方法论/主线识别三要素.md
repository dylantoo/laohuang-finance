# 主线识别三要素

> 判断一个板块能否成为阶段性主线

---

## 三要素

### 1️⃣ 有催化

按含金量排序：
```
政策驱动 > 行业景气反转 > 技术突破 > 涨价 > 事件驱动
```

### 2️⃣ 资金认

| 指标 | 标准 |
|------|------|
| 板块成交额 | 连续3天以上放大 |
| 涨停数 | 每天3-5只以上 |
| 龙头涨幅 | 至少翻倍或接近翻倍 |

### 3️⃣ 有梯队

```
龙头 → 中军 → 跟风 → 补涨
```

## 3问速查

1. 有催化吗？（政策/行业/技术）
2. 资金认吗？（放量+涨停）
3. 有龙头带吗？（龙头+跟风）

> 少一条都不算真正的主线。
