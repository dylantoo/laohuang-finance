# 回调 vs 退潮五维校验法

> 当板块涨了30%+后出现回调时，判断是上车机会还是陷阱

---

## 五维校验流程

```
发现涨30%+的板块回调
         │
         ▼
 第一步：回调缩量了吗？
    ┌──┴──┐
    是    否 → ❌ 退潮
    │
    ▼
  第二步：破EMA20了吗？
    ┌──┴──┐
    没破   破了 → ❌ 退潮
    │
    ▼
  第三步：板块内还有涨停吗？
    ┌──┴──┐
    有    没有 → ❌ 退潮
    │
    ▼
  第四步：核心逻辑还在吗？
    ┌──┴──┐
    在    不在 → ❌ 退潮
    │
    ▼
  ✅ 上车机会
```

> 五维中三个以上健康回调 → 上车。两个以下 → 放弃。

---

## 各维度详解

### 维度一：回调的"姿势"
- ✅ 健康：缩量回调，不破EMA10/EMA20，3-5天
- ❌ 退潮：放量下跌，砸穿EMA20，超过7天

### 维度二：看"兄弟"
- ✅ 健康：中军不跌，还有涨停
- ❌ 退潮：龙头一跌全部跟跌

### 维度三：看大盘
- ✅ 健康：大盘没破位
- ❌ 退潮：大盘涨但板块跌

### 维度四：看逻辑
- ✅ 健康：核心催化还在
- ❌ 退潮：利好出尽/逻辑变了

### 维度五：看阶段
- 第一次回调 → 最安全
- 第三次回调 → 大概率顶部
