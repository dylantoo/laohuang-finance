## 主线识别三要素

一个板块要成为阶段性主线，三条腿必须同时站稳。

### 1️⃣ 有催化（硬逻辑）

按含金量排序：
```
政策驱动 > 行业景气反转 > 技术突破 > 涨价 > 事件驱动
```

### 2️⃣ 资金认

| 指标 | 标准 |
|------|------|
| 板块成交额 | 连续3天以上放大 |
| 板块内涨停数 | 每天至少3-5只涨停 |
| 龙头涨幅 | 至少翻倍或接近翻倍 |

**关键信号**：板块成交额占全市场比例持续提升（5%→15%+）。

### 3️⃣ 有梯队

```
龙头（1-2只，人气最旺）
  └─ 中军（大市值机构票，压舱石）
      └─ 跟风（3-5只跟涨）
          └─ 补涨（N只，龙头高位后扩散）
```

> 如果只有一只票在涨 → 这不是主线，是妖股。

## 回调 vs 退潮五维校验法

板块涨了30%+后出现回调，用五维法判断：

### 维度一：回调的"姿势"

| 特征 | ✅ 健康回调（上车） | ❌ 退潮回调（陷阱） |
|------|------------------|------------------|
| 幅度 | 不破EMA10，最多到EMA20 | 砸穿EMA20，碰EMA55 |
| 量能 | 缩量（成交降30-50%） | 放量下跌 |
| 天数 | 3-5天，不拖泥带水 | 超过7天还在阴跌 |
| 龙头 | 跌但没跌停，跌幅<5% | 跌停或连续大跌 |
| K线 | 每天有下影线 | 光头阴线 |

> **口诀**：缩量回调不破EMA10是爹，放量下跌砸穿EMA20是雷。

### 维度二：看"兄弟"

- ✅ 健康回调：中军不跌，还有别的涨停，补涨股活跃
- ❌ 退潮回调：龙头一跌全部跟跌，无一幸免

### 维度三：看大盘"脸色"

- ✅ 大盘没破位或上涨 + 板块回调 → 正常
- ❌ 大盘涨但板块跌 → 资金在流出

### 维度四：看催化剂"成色"

- ✅ 逻辑还在 = 回调（上车机会）
- ❌ 逻辑变了/利好出尽 = 退潮（不能上车）

### 维度五：看主升结构走了多远

- 第一波30%后第一次回调 → ✅ 最安全
- 第二波后第二次回调 → ⚠️ 次安全
- 第三波后第三次回调 → ❌ 大概率顶部

> 五维中三个以上健康回调 → 上车。两个以下 → 放弃。

## 混沌期生存法则

### 混沌期特征

```
板块轮动速度：2-3天一个
龙头封板率：＜50%，二板封不住
赚钱效应：今天涨明天跌，追高必套
成交量：缩量，不放量
```

### 三不 + 三可

| ❌ 不做 | ✅ 做 |
|--------|------|
| 不追高 | 轻仓试错（≤30%） |
| 不满仓 | 埋伏等风口 |
| 不做T（纪律） | 低吸不追涨 |

### 混沌期仓位

- 总仓位：≤30%
- 单票上限：10%
- 止损：-5%（比主升更严）
- 现金：≥70%

### 混沌期有赚就跑

- 浮盈5-8% → 止盈
- 浮亏-5% → 止损
- 不指望吃一整段主升

### 核心心态

> 我不是在赚钱，我是在活着等下一个主线的到来。

## 进场时机与节奏

### 早期发现（第1-3天）

```
第1-3天：板块放量，龙头涨停
→ 小仓位10-15%，买龙头
→ 第2天龙头继续封板 → 不是一日游

第4-7天：板块梯队形成
→ 加仓到30-40%
→ 板块回调EMA10企稳 → 最佳加仓点
```

### 已涨30%+ → 等回调

```
❌ 连续大涨后追入
✅ 第一次回调EMA10不破+缩量企稳
✅ 龙头放量下跌后的缩量企稳
✅ 补涨股出现 → 资金还在扩散
```

## 离场信号分级

### 🟢 初级（警惕，不减仓）
- 板块涨了1.5-2个月以上
- 龙头开始滞涨
- 涨停数从10+降到3-5只

### 🟡 中级（减仓50%）
- 龙头第一次大跌但不反包
- 补涨股乱飞，龙头不动了
- 板块成交额天量后缩量

### 🔴 终极（必须走）
- 龙头跌停
- 板块跌破20日均线
- 出现新的替代主线
- 龙头回撤超-20%且站不回去

> ⚠️ **最重要信号**：补涨股乱飞 + 龙头停滞 = 主升浪尾声

## 仓位管理全表

| 阶段 | 特征 | 仓位 |
|------|------|------|
| 主升期 | 主线清晰，龙头涨，梯队完整 | 70-100% |
| 震荡期 | 主线分化，指数横盘 | 50% |
| 混沌期 | 无主线，热点散乱 | ≤30% |
| 退潮期 | 老主线大跌，无新主线 | 0-20% |

### 止损铁律

- 单票-10%建议止损，-15%无条件止损
- 总仓-15%清仓制动
- 混沌期止损-5%

## 123冷冻期硬规则

> 专门针对追涨被套的问题

```
板块涨30%+开始回调时：
1️⃣ 第一天 → 别动，观察怎么跌
2️⃣ 第二天 → 继续看，会不会反包
3️⃣ 第三天 → 跌到EMA20企稳了 → 考虑上车
🚫 前3天绝对不买！
```

## 混沌期择方向——四不碰三要

### ❌ 四不碰
1. 昨天大涨的（明天大概率跌）
2. 已经涨了2周的（可能尾部）
3. 游资票、纯概念（说走就走）
4. 业绩差、估值虚高（跌最狠）

### ✅ 三要（潜伏候选）
1. 有政策底、产业逻辑的
2. 板块跌了3-5天、缩量到地量的
3. 龙头走势独立、没随板块大跌的

### 排雷法选方向

```
Step 1：列出近期活跃板块（10-15个）
Step 2：剔除最近3天大涨的 → 8-10个
Step 3：剔除炒概念的 → 5-6个
Step 4：选出连跌3天+缩量企稳的 → 2-3个
Step 5：选逻辑最硬的1个 → 试仓
```

## 每日收盘检查清单

```
□ 今天有没有新板块涨3%+？什么逻辑？
□ 最近3天涨最多的是什么？→ 明天不追
□ 最近3天跌最多的是什么？→ 缩量了吗？逻辑在吗？
□ 今天总仓位多少？→ 超过30%了？减
□ 有没有票浮亏超过-5%？→ 明天处理
```
