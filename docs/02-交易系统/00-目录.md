## 核心原则

1. **周期优先** — 判断市场阶段是第一要务
2. **主线思维** — 只做主线板块，不做杂毛
3. **纪律为王** — 做T有前提，止损不犹豫
4. **多元思维** — 交叉验证判断
5. **风控底线** — 单票-10%、总仓-15%无条件止损
