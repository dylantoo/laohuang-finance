# 做T决策框架

> 做T必须有前提条件

---

## 做T触发条件

**早上低开（绿开）时**才考虑做T

## 做T前提（必须同时满足）

1. ✅ 主线还在（未退潮）
2. ✅ 大盘没破位
3. ✅ 市场处于主升期或震荡期

## ❌ 什么时候不做T

- 混沌期 → **不做T**，绿了也不接（可能是下跌中继）
- 退潮期 → **不做T**
- 个股不在主升/震荡 → **不做T**

## 操作方式

- 低开时买入 → 涨超5%时卖出
- 只用机动仓位做T
- 底仓不动
- 不为做T去买新票
