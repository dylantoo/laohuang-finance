# 仓位管理规则

> 四阶段仓位 + 波动率联动调仓

---

## 四阶段仓位

| 阶段 | 仓位 | 策略 |
|------|------|------|
| **主升期** | 70-100% | 重仓龙头，持仓不动 |
| **震荡期** | 50% | 保留最强，换掉弱的 |
| **混沌期** | ≤30% | 强制降仓，空仓也行 |
| **退潮期** | 0-20% | 清仓或极小仓位试错 |

## 单票仓位

- 1只龙头：50-70%
- 2只：各30-40%
- 3只（上限）：各20-30%

## 波动率联动调仓

| 波动率 | 系数 | 动作 |
|--------|------|------|
| <15%（低） | ×1.25 | 可加仓 |
| 15-25%（正常） | ×1.0 | 标准 |
| 25-35%（高） | ×0.75 | 减仓 |
| >35%（极高） | ×0.5 | 强减仓 |

> 例：震荡期50%×0.75（高波动）= 实际约37.5%

## 混沌期特殊仓位

| 规则 | 数值 |
|------|------|
| 总仓位 | ≤30% |
| 单票上限 | 10% |
| 止损 | -5% |
| 现金 | ≥70% |
