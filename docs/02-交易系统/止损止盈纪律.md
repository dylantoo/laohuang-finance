# 止损止盈纪律

> 铁律 + 离场信号分级

---

## 止损铁律（不变）

| 情况 | 操作 |
|------|------|
| 单票浮亏-10% | 建议止损 |
| 单票浮亏-15% | **无条件止损** |
| 总仓回撤-15% | **无条件清仓制动** |
| 混沌期-5% | 止损（更严） |

## 持仓周期

### 盈利状态
- 中线持有，让利润奔跑
- 止盈参考：**2-4周或新高不再持有**
- 超过一个月没新高 → 警惕

### 亏损状态（双重触发机制）
- ⏰ **时间触发**：第3天警觉，第5天必须决策
- 🚨 **信号触发**（优先于时间）：
  - 主线退潮 → 立即走
  - 大盘破位 → 立即走
- **信号 > 时间**

## 离场信号分级

### 🟢 初级（警惕）
- 板块涨了1.5-2个月
- 龙头开始滞涨
- 涨停从10+降到3-5只

### 🟡 中级（减半仓）
- 龙头大跌不反包
- 补涨股乱飞，龙头不动
- 板块天量后缩量

### 🔴 终极（必须走）
- 龙头跌停
- 板块破20日线
- 替代主线出现
- 龙头回撤-20%且站不回

> ⚠️ **最重要信号**：补涨股乱飞+龙头停滞=主升浪尾声
