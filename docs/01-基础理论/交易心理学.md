# 交易心理 — 《股票大作手回忆录》精华

> 适用场景：每日复盘、做T决策、心态管理。和 stock-pivot 的止损/止盈规则配合使用。

> 核心人物：杰西·利弗莫尔（Jesse Livermore），20世纪最伟大的交易员之一

## 一、核心理念

**市场永远是对的，错的只有你的判断。**

交易不是预测未来，而是**管理概率**。没有人能每次都判断对，但任何人都可以通过纪律让整体获利。

## 二、做T心法（与老黄的做T策略直接对应）

### 1. 趋势是你的朋友，永远不要和趋势作对
> "我赚大钱从来不是因为我的想法，而是因为我坐着不动。"

**应用**：
- 主线在+上升趋势 → 做T加仓，别因为恐高而卖飞
- 主线退潮+下跌趋势 → 别抄底，做T只做反T（先卖后买）
- "坐着不动"不是不动，是**在主线上不动，在退潮时跑得比谁都快**

### 2. 不要向下摊平（金字塔加仓法）
> "第一次亏损是最便宜的亏损。"

**老黄版**：主线在的时候，回调到支撑位加仓 → ✅ 正确
**错误示范**：股票跌了觉得"便宜了"加仓，趋势还是空的 → ❌ 接飞刀

**正确做法**：
- 主线确认+回调到EMA55支撑 → 金字塔式加仓（越跌越买但量递减）
- 主线退潮+任何回调 → 不买，等企稳

### 3. 耐心等待关键点，不到不入
> "不是天天都有行情。赚大钱的人，大部分时间都在等待。"

**应用**：
- 早上开盘前15分钟不交易（波动最大）
- 等一只票的关键转折信号出现再动手
- 做T交易频率：**一天最多2-3次T，不要频繁交易**

### 4. 利润会自己照顾好自己，亏损不会
> "亏损的仓位应该剪掉。不要让损失变成大窟窿。"

**对应stock-pivot**：止损-10%建议/-15%无条件。严格执行。
**重点**：止损不是"猜错了"的惩罚，是**交易计划的一部分**。

## 三、四种交易者心态陷阱

### 陷阱1：希望 & 恐惧
| 情况 | 正常反应 | 正确做法 |
|------|---------|---------|
| 买了涨了 | 怕回吐想卖 | **让利润奔跑，按规则止盈** |
| 买了跌了 | 希望反弹想等 | **按规则止损，不管后面涨不涨** |
| 主线确认 | 犹豫怕追高 | **该重仓时就要重仓** |
| 主线退潮 | 不舍得走 | **该空仓时就要空仓** |

**口诀**：买在犹豫（主线刚确认时），卖在一致（人人都说好的时候）。

### 陷阱2：报复交易
> "赔钱后想一把赚回来，就会犯更大的错误。"

**应用**：
- 当天连续做T止损2次 → **立即停手，明天再说**
- 连续亏损3天 → 空仓一周，复盘找原因
- 千万别"补仓拉低均价"→ 那是懦夫的做法，不是交易

### 陷阱3：因小失大
> "为了一两个点的差价错过了一个大行情。"

**做T的版**：
- 做T赚了1%就想卖，结果卖飞了后面20%的行情
- **做T的仓位和中长线仓位要分开**，不要拿底仓做T

**建议**：
- 底仓不动（中长线持有）
- 机动仓做T（不超过总仓位的30%）
- T飞了不追，等下一个信号

### 陷阱4：听了"内部消息"
> "内幕消息是最贵的消息。"

**应用**：
- 任何人跟你推荐股票 → 按你的系统重新分析，不要盲信
- 包括我的推荐 → 你看，我今天推了中兴通讯，你也要自己判断
- **最终决策权在你**

## 五、做T专用心法

### 日内做T信号

| 信号 | 操作 | 概率 |
|------|------|:----:|
| 开盘急跌3%+但板块不弱 | 低吸做正T | 高 |
| 开盘急拉3%+但没量 | 高抛做反T | 高 |
| 早盘缩量横盘 | 不动，等方向 | - |
| 下午2:30后放量拉升 | 可追一点正T | 中 |
| 下午2:30后放量下跌 | 跟出 | 中 |

### 浮盈加仓原则
> "只有第一笔仓位盈利了，才加第二笔。浮亏了不但不加，还要减。"

**做T的节奏**：
- 第一笔T盈利 → 可以做第二笔T
- 第一笔T亏了 → 今天不再T了

## 七、血泪教训案例

### 利弗莫尔的破产经历
- 一生破产4次，每次都是因为**违反了自己的规则**
- 赚了几百万又亏光，因为他"觉得自己天下无敌了"
- 最终他学会：**敬畏市场**

### 老黄版：常见死法
1. 主线还在但恐高卖了 → 踏空
2. 主线退了但还想"做T" → 被套
3. 一只票亏了想"等反弹" → 越等越深
4. 连续对了3次就觉得自己是神 → 第四把梭哈亏回去

**解法**：**每次盈利后，提醒自己"我只是运气好"。**
