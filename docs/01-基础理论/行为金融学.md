## 二、炒股最常见的7种认知偏误

### 1. 锚定效应 — 最常见的亏钱陷阱
> "先入为主的价格印象会绑架你的判断。"

**表现**：
- 某股票曾经100块，现在70块 → "便宜了"（其实是合理的70块，不是便宜）
- 涨停板挂了卖单 → 因为"涨停价是锚"
- 被套后不卖 → 因为"回本"（成本价是锚）

**解法**：
- 不看买入成本，只看**当前趋势和止损位**
- 问自己：如果我现在没有持仓，这个价格我会买吗？
- stock-pivot的止损规则就是破锚定效应的

### 2. 确认偏误 — 你只看到你想看到的
> "持有某只票后，会自动找支持它的理由，忽略反面证据。"

**表现**：
- 买了中兴通讯后，到处找"5G利好"的信息
- 看到"中兴跌了"的新闻，觉得"那是技术性调整"
- 不看"中兴被制裁风险"的报告

**解法**：
- 强制自己列出**3个看空的理由**
- 用stock-pivot的规则做判断，不要用"我觉得"
- 每天看跌幅榜比看涨幅榜更有价值

### 3. 损失厌恶 — 亏100的痛苦 > 赚100的快乐
> "同样是100块，亏损的痛苦强度是盈利快乐的两倍。"

**表现**：
- 盈利10%就想卖（怕利润回吐）
- 亏损10%死扛（不肯认输）
- 做T赚了1%快乐，亏了0.5%痛苦一整天

**解法**：
- 按规则止损，**不带感情**
- 赚钱了让利润奔跑（>2周或新高再走）
- 把亏损看成"交易成本"不是"失败"

### 4. 近期偏误 — 最近涨了就以为会一直涨
> "上涨趋势中，人会自动外推：昨天涨、今天涨、明天也会涨。"

**表现**：
- 连续3根阳线就觉得"牛市来了"
- 涨停板追进去 → 第二天跌了 → "只是回调" → 继续跌 → 慌了
- 忘记看周期位置，只看最近3天的K线

**解法**：
- 看周K和月K，不要只看日K
- 用stock-pivot的EMA趋势过滤短期噪音
- 判断是主升浪还是反弹（反弹不做）

### 5. 沉没成本谬误 — "已经亏了这么多了……"
> "过去花掉的钱、时间，不应该影响未来的决策。"

**表现**：
- 一只票亏了30%，想"都亏这么多了，现在卖了就是韭菜"
- 想"回本就卖" → 结果从30%亏到50%
- 越等越深，最后割在最低点

**解法**：
- 沉没成本**不是成本**，未来走势才重要
- stock-pivot：亏损-10%建议/-15%无条件走
- 问：如果现在给我现金，我会买这只票吗？→ 不会 → 卖

### 6. 过度自信 — 连续对几次就飘了
> "赚了钱的人最容易觉得自己是天才。"

**表现**：
- 连续做T对了3次 → 加大仓位 → 第4次全亏回去
- 推荐票涨了 → 觉得自己"看得很准" → 下一个票梭哈
- 频繁交易，手停不下来

**解法**：
- 盈利后**提现一部分**
- 连续成功3次+ → 停手1天
- 告诉自己：**市场只是暂时奖励了我的错误**

### 7. 从众效应 — 别人买我也买
> "当所有人都买同一个东西时，价格通常已经贵了。"

**表现**：
- 看到群里/论坛都在吹某只票 → 控制不住想买
- "北向资金买了某某" → 跟着买
- "主力资金流入" → 跟

**解法**：
- 用你的系统分析，**不跟风**
- 人人都在聊的票 → 大概率已经高了
- 真正的机会没几个人知道
